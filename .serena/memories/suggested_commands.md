Windows/PowerShell commands for this project:
- Install deps: npm ci
- Dev server: npm run dev
- Build: npm run build
- Start production server: npm run start
- Lint: npm run lint
- Tests: npm run test
- Watch tests: npm run test:watch
- Coverage: npm run test:coverage
- Prisma generate: npx prisma generate
- Prisma migrate deploy: npx prisma migrate deploy
Useful shell commands on Windows:
- List files: Get-ChildItem
- Recursive file search: Get-ChildItem -Recurse
- Content read UTF-8: Get-Content -Encoding UTF8 <path>
- Text search (preferred): rg <pattern>
- Git status: git status
- Git diff: git diff
